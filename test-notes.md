# 動作確認メモ

## 問題点
1. ホーム画面の「試行数: 回」の表示 - 数値が表示されていない（configのtotalTrialsが空になっている可能性）
   - 実際は「試行数: 回」と表示されている → localStorageの古い設定が読み込まれている
   - loadConfigのマイグレーションロジックで対応済みだが、DBから読み込んだ設定が古い形式の可能性

2. 実験画面で「0 /」と表示 - totalTrialsが0になっている
   - 同じ原因: DBに保存された古い設定にtotalTrialsがない

3. 練習モードの表示がない - カウントダウン後すぐに本番が始まった
   - practiceTrialsが0になっている可能性

## 対応
- DBの古い設定にもマイグレーションロジックを適用する必要がある
- ExperimentContextでDBから読み込んだ設定にもデフォルト値をマージする
